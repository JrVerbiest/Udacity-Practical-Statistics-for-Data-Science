| CRITERIA                                   | MEETS SPECIFICATIONS                                         |

# Environment Setup

- Create environment: 
```
$ conda env create -f conda.yml
```

- Activate environment: 
```
$ conda activate udacity_venv

```
$ conda env update --name udacity_venv --file conda.yml --prune
